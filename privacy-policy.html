Политика конфиденциальности

Что бот обрабатывает, зачем и как вы можете управлять своими данными.

Настоящая Политика конфиденциальности описывает, какие данные может обрабатывать Discord-бот ParadiseBot (далее — «Бот») и как эти данные используются для работы функций Бота.

01

Какие данные мы обрабатываем

Для работы функций Бот может получать, обрабатывать и хранить следующие данные Discord:

  • Идентификаторы: ID пользователя, сервера, канала, сообщения, роли, розыгрыша и других объектов Discord;
  • Публичные данные профиля: имя пользователя, никнейм на сервере, аватар, роли и статус присутствия, если Discord передаёт эти данные Боту;
  • Активность: сведения о сообщениях для подсчёта активности, рейтинга, квестов и допуска к ивентам, включая время, сервер, канал и ID сообщения;
  • Розыгрыши: данные о созданных розыгрышах, участниках, победителях, призах, подтверждениях выигрыша и Random.org-проверках;
  • Команды и настройки: параметры команд, настройки сервера, настройки администраторов, данные авторизации администратора и выбранные пользователями параметры;
  • Модерация: содержимое сообщений может временно анализироваться для фильтрации запрещённых фраз и оскорблений администрации;
  • Интеграции: данные, необходимые для музыкальных функций, AI-команд, поиска тиммейтов, Valorant-трекера, Spotify, Яндекс Музыки и других включённых функций;
  • Технические данные: журналы ошибок, служебные события, ID взаимодействий Discord и данные, необходимые для диагностики работы Бота.
02

Как мы используем данные

Данные используются только для работы функций Бота:

  • проведение розыгрышей, выбор победителей и публикация Random.org-подтверждений;
  • подсчёт активности, рейтингов, квестов и профилей пользователей;
  • автоматическая модерация сообщений и защита администрации сервера от оскорблений;
  • работа административных команд, верификации, приватных голосовых каналов, поиска тиммейтов и музыкальных команд;
  • обработка AI-запросов и интеграций, если они включены на сервере;
  • поддержка безопасности, диагностики ошибок и стабильности Бота.

Мы не продаём пользовательские данные и не используем их для сторонней рекламы.

03

Содержимое сообщений

Бот может получать содержимое сообщений через Discord Message Content Intent, если это необходимо для команд, фильтров модерации, подсчёта активности или AI-функций. ParadiseBot не предназначен для хранения полной истории чата. Для активности обычно сохраняются минимальные служебные данные, такие как ID пользователя, сервера, канала, сообщения и время события.

04

Сторонние сервисы

Для отдельных функций данные могут передаваться сторонним сервисам только в объёме, необходимом для выполнения конкретной функции:

  • Discord API — основная работа Бота внутри Discord;
  • Random.org — генерация и проверка случайных результатов розыгрышей;
  • AI-провайдеры — обработка AI-команд, если AI-функции включены;
  • музыкальные сервисы и Lavalink-узлы — воспроизведение музыки;
  • Spotify, Яндекс Музыка, Valorant-трекер и другие интеграции — только при использовании соответствующих функций.
05

Хранение данных

Данные хранятся в базе данных Бота столько, сколько это необходимо для работы функций, администрирования сервера, истории розыгрышей, подтверждения победителей, рейтингов и технической диагностики. Администрация Бота может удалять или очищать данные при переносе, обслуживании, удалении Бота с сервера или по обоснованному запросу.

06

Доступ и удаление данных

Вы можете запросить просмотр, исправление или удаление связанных с вами данных. Для этого обратитесь к владельцу ParadiseBot (контакт указан ниже) или к администрации Discord-сервера, где используется Бот. Запрос может быть отклонён или ограничен, если удаление данных противоречит требованиям закона, безопасности, расследованию нарушений или технической целостности розыгрышей.

07

Безопасность

Мы принимаем разумные меры для защиты данных, включая ограничение доступа к базе данных, токенам и административным функциям. Однако ни один способ передачи или хранения данных не может быть полностью безопасным, поэтому абсолютная безопасность не гарантируется.

08

Дети

Бот не предназначен для пользователей младше минимального возраста, установленного Условиями использования Discord для вашего региона. Мы сознательно не собираем данные таких пользователей.

09

Изменения политики

Настоящая Политика может обновляться. Изменения вступают в силу с момента публикации обновлённой версии на данной странице.
